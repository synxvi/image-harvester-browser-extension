# Image Hover Save - Chrome 图片悬浮保存扩展

一款简洁高效的 Chrome 浏览器扩展，只需将鼠标悬停在图片上即可快速下载。悬停后出现下载按钮，一键保存到本地下载文件夹。

## ✨ 基础功能

| 功能 | 说明 |
|---|---|
| **🖱️ 悬浮触发** | 鼠标悬停在图片上，自动弹出下载按钮 |
| **⏱️ 可调延迟** | 按钮出现延迟可在 0.5s ~ 3s 之间自定义调节（默认 1 s）|
| **💾 一键下载** | 点击按钮直接保存到默认下载目录，无弹窗确认 |
| **🎬 视频支持** | 支持视频元素（`<video>`）的悬浮下载 |
| **🔌 开关控制** | 一键启用/禁用扩展 |
| **🌐 站点屏蔽** | 可排除不想生效的网站域名 |

## 🚀 增强功能

### 多路径批量保存
- 开启后可为每个路径配置独立的子文件夹
- 每个路径在图片上显示为独立的悬浮按钮
- 一次悬停即可选择多个目标路径分别下载
- 支持路径排序（上下调整顺序）
- 仅在普通下载模式下可用

### 基础保存目录
- 配置一个公共父目录，所有下载子目录统一在其下
- 例如设置 `图片` 为基础目录，多路径 `风景/人物` 会自动保存为 `图片/风景/` 和 `图片/人物/`

### 批量下载
- **📦 图库模式**：在新标签页中打开页面所有图片的图库视图
- **🗜️ ZIP 打包**：将页面所有图片打包为 ZIP 文件一次性下载

### 实验性下载模式
针对防盗链等限制性网站提供多种备用方案：

| 模式 | 说明 |
|---|---|
| **普通模式** | 通过后台 Worker 下载，支持进度条显示，兼容性最好 |
| **Fetch 模式** | 通过当前页面上下文发起请求绕过部分热链限制 |
| **Canvas 提取** | 将图片绘制到 Canvas 再导出，适合特殊渲染场景 |

### 高级检测类型
- **SVG 元素** — 检测并支持 SVG 图片下载
- **背景图片** — 检测 CSS 背景图并支持下载

### 视觉反馈
鼠标悬停在可识别的媒体上时，可选边框高亮提示：
- 无高亮 / 灰色边框 / 绿色边框 三档可选

### 其他高级选项
| 功能 | 说明 |
|---|---|
| **最小尺寸过滤** | 只检测大于指定尺寸的图片（50~1000px），避免误触小图标 |
| **文件格式过滤** | 自定义允许的文件扩展名列表（逗号分隔）|
| **WebP 转 PNG** | 自动将 WebP 格式转换为 PNG，提升兼容性 |
| **长隐藏延迟** | 将按钮消失延迟延长至 1.5s，适配问题视频播放器 |
| **右键菜单** | 针对纯链接（非 `<img>` 标签）提供右键下载入口 |

## 📦 安装方式

### 方式一：Chrome Web Store 安装（推荐）

[前往 Chrome Web Store 安装](https://chromewebstore.google.com/detail/image-hover-save/lhmljebjlhdafkminimimaokjliabegg)

> ⚠️ Web Store 版本可能不是最新版本，如需最新功能请使用方式二。

### 方式二：手动加载（开发者模式）

1. 克隆或下载本项目代码
2. 打开 Chrome，访问 `chrome://extensions/`
3. 右上角开启「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择项目中的 `extension` 文件夹

## 🎮 使用方法

1. 在任意包含图片的网页中浏览
2. 将鼠标悬停在目标图片上（默认等待 1.5 秒）
3. 图片右上角会出现下载按钮 💾
4. 点击按钮即可将图片保存到下载文件夹
5. 如已配置多路径，会显示多个按钮对应不同路径

点击扩展图标可随时调整设置、切换模式或禁用扩展。
